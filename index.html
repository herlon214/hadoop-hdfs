HDFS

O que é?

O HDFS é um projeto da Apache Software Foundation e um subprojeto do projeto Apache Hadoop.

O Hadoop foi criado para armazenar grandes quantidades de dados, de terabytes e pentabytes, e o HDFS é o sistema de armazenamento dele.

Qual problema ele veio resolver?

Portabilidade: Funciona em vários sistemas operacionais.

Escalabilidade: Armazenamento de grandes quantidades de dados distribuidos entre os clusters.

Tolerância a falhas: Recuperação de arquivos rápida e automática.

HDFS na prática

Configuração

Um cluster no Yahoo! possui em torno de 3500 nós HDFS.
Cada cluster possui:

2 quad core Xeon processors @ 2.5ghz
Red Hat Enterprise Linux Server Release 5.1
Sun Java JDK 1.6.0_13-b03
4TB SATA drives
16GB RAM
1-gigabit Ethernet

70% (2.8TB) do disco é reservado exclusivamente para o HDFS.
O resto (1.2TB) para o S.O, logs etc.
